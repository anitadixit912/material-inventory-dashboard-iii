# Stock Advisor Agent

An AI agent that analyzes material stock levels and provides replenishment recommendations based on real-time inventory data.

## Overview

Uses A2A Protocol, LangGraph, LiteLLM, and SAP Cloud SDK.

## Structure

- `app/main.py` - A2A server entry
- `app/agent_executor.py` - Request handling
- `app/agent.py` - Agent logic
